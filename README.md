# Neura-VIZ

Небольшой визуализатор для робототехнического соревнования "Decode".

Описание
-
`Neura-VIZ` — простая веб-visualizer-реализация игрового поля, роботов и образцов.
Проект показывает позицию робота(ов), траектории выпущенных образцов и корзины (goals).

Ключевая правка
-
В этой версии добавлена логика ориентации поля в зависимости от альянса. Для `red` альянса поле переворачивается на 180° так, чтобы водитель красного альянса видел свою цель спереди (т.е. изображение инвертируется для удобства управления). Это затрагивает:
- визуализацию корзин (baskets),
- положение и ориентацию роботов,
- отображение образцов и траекторий,
- историю движения робота.

Установка
-
Требуется Node.js (рекомендуется 18+)

1. Установите зависимости:

	npm install

2. Запустите в режиме разработки:

	npm run dev

Проект использует Vite — после запуска откройте указанный локальный адрес (обычно http://localhost:5173).

Использование
-
- Код визуализатора находится в `components/Field.tsx`.
- Альянс задаётся из состояния приложения (проп `alliance: 'blue' | 'red'`). Для теста можно переключить значение в месте, где `Field` используется, чтобы увидеть переворот поля.

Разработка
-
- Структура проекта:
  - `components/` — React-компоненты (включая `Field.tsx`)
  - `hooks/` — пользовательские хуки
  - `utils/` — утилиты
